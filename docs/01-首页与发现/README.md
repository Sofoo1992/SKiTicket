## 核心价值

1. **高效分发**: 通过金刚区快速引导用户进入"消费"场景 (买票、约课)。
2. **场景细分**: 通过 Feed 流标签 (拼车、拼房、约搭子) 解决滑雪特定痛点。
3. **运营灵活**: 首页核心展位 (Banner/入口/推荐) 均支持后台动态配置，适应雪季/非雪季的运营策略切换。
