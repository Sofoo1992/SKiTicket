# 内容发布流程

## 1. 发布入口
- 底部导航栏中央悬浮按钮 **"+ 我要发布"**。
- 点击后弹出类型选择菜单，选择 **"发布动态"**。

---

## 2. 媒体选择与编辑
支持图文混排，鼓励高质量内容生产。

### 限制规则
- **总数量限制**: 最多选择 **18** 个媒体文件 (图片+视频)。
- **视频限制**: 最多包含 **6** 个视频。
- **文件要求**:
  - 图片: JPG/PNG/HEIC 格式，单张 ≤ 20MB。
  - 视频: MP4/MOV 格式，时长 ≤ 3分钟，大小 ≤ 500MB。

### 交互功能
- **拍摄/相册**: 支持实时拍摄或从相册多选。
- **拖拽排序**: 长按已选媒体可拖拽调整顺序。
- **封面设置**: 对视频内容可单独设置封面图。

---

## 3. 内容编辑

### 文本域
- **标题 (选填)**: 吸引眼球的短标题。
- **正文 (必填)**: 核心内容描述。
  - **字数限制**: 最大 **1000** 字。
  - **输入体验**: 支持换行，Textarea 高度自适应。

### 扩展信息
- **所在位置 (LBS)**:
  - 默认: 自动定位当前位置。
  - 逻辑: 优先展示"滑雪场"类型的 POI。
  - 列表: 最近使用位置 + 热门雪场推荐。
- **隐私设置**:
  - **谁可以看**: 公开 (默认) / 仅自己 / 仅粉丝。

---

## 4. 发布与草稿

- **发布状态**:
  - 如果媒体正在上传中，"发布"按钮置灰并显示进度。
  - 上传完成后，按钮高亮可点击。
- **草稿箱**:
  - 用户如果点击返回或非法退出，提示"是否保存草稿"。
  - 再次进入发布页时，提示"是否恢复上次编辑内容"。
