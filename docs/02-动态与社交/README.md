## 核心设计

- **重媒体**: 支持多达 18 张图/6 个视频，鼓励"大片"分享。
- **LBS 优先**: 强关联地理位置 (雪场)，方便基于同雪场的社交连接。
- **结构化标签**: 通过结构化标签 (如约拍、拼车) 提高信息匹配效率。
