## 核心特性

- **结构化成色**: 统一定义"#9成新"等标准，减少沟通成本。
- **灵活物流**: 支持快递与线下 (雪场面交) 两种模式，适应滑雪场景。
- **信任机制**: 基于社区动态和信用的交易体系。
