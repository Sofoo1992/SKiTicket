# 雪哒 APP 产品文档

> 🎿 户外滑雪社交平台 —— 连接雪友，记录精彩

---

## 🌟 产品定位

**雪哒**是一款面向滑雪爱好者的户外社交 APP，提供：

- 🏔️ **雪场发现与导航** — 全国 200+ 雪场信息
- 📱 **动态社交** — 雪友分享、互动、关注
- 🗺️ **轨迹记录** — GPS 滑行轨迹、数据统计
- 🎫 **票务购买** — 雪票、住滑套餐、教练课程
- ♻️ **闲置交易** — 二手装备买卖
- 🎉 **活动赛事** — 约伴、赛事、滑雪营
